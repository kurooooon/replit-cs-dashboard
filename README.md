# Customer Support Admin Dashboard

カスタマーサポート管理用のダッシュボードアプリケーションです。

## 機能

- 管理者認証
- ユーザー検索機能
- 詳細なユーザー情報の表示
- クリーンなインターフェース

## 技術スタック

- Frontend: React + TypeScript
- Backend: Express.js
- スタイリング: Tailwind CSS + shadcn/ui
- 状態管理: TanStack Query

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## デフォルトの管理者アカウント

- ユーザー名: admin
- パスワード: admin123
